# Sentiment Analysis with NLP (FinBERT)

## 📌 Overview
This project applies Natural Language Processing (NLP) techniques to analyze the sentiment of financial news and evaluate its impact on stock price movements, using real-world financial data.

---

## 🧠 Problem
Understand whether the sentiment of financial news can help explain or anticipate market behavior.

---

## 📊 Data
- Financial news headlines (NewsAPI)
- Stock price data (yFinance)
- Data processed and stored using BigQuery

---

## ⚙️ Solution
- Text preprocessing and cleaning
- Sentiment classification using FinBERT (Hugging Face)
- Integration with structured financial data
- Regression model using XGBoost

---

## 📈 Results
- Identified weak correlation between sentiment and stock price movements
- Observed high noise in financial data
- Demonstrated that sentiment alone is not sufficient for prediction

---

## 🧪 Key Learnings
- Practical application of NLP in finance
- Importance of combining structured and unstructured data
- Limitations of predictive models in real-world scenarios

---

## 🛠️ Tech Stack
- Python
- Transformers (FinBERT)
- Pandas
- BigQuery
- XGBoost

---
